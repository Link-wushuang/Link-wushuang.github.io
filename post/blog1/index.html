Hugo + GitHub 个人博客搭建教程

从零开始，手把手教你使用 Hugo 静态网站生成器搭建个人博客，包括环境准备、博客创建、主题配置等完整流程

一、环境准备

在开始搭建博客之前，我们需要准备两个必要的工具：Git 和 Hugo。

1.1 安装 Git

Git 是一个版本控制工具，用于管理我们的博客源码。

下载步骤：

  1. 访问 Git 官网
  2. 下载适合你操作系统的安装程序
  3. 运行安装程序，保持默认设置，一路点击"下一步"即可

安装完成后，打开命令行输入 git --version，如果显示版本号则说明安装成功。

1.2 下载 Hugo

Hugo 是一个用 Go 语言编写的静态网站生成器，以速度快、配置简单著称。

下载步骤：

  1. 访问 Hugo Releases 页面
  2. 选择适合版本的 hugo_extended 版本（Extended 版本支持 SCSS/SASS）
  3. 根据你的操作系统选择对应的压缩包下载

我选择的是 hugo_extended_0.159.1_windows-amd64.zip，至于下面的withdeploy貌似是可以更高效的进行配置和部署的，但是我不太会用，所以还是算了吧。。。

  1. 下载好压缩包后，先在D盘新建一个hugo文件夹，再在hugo文件夹里再新建一个bin文件夹

  2. 接下来将hugo压缩包解压至bin目录下

  3. 打开环境变量管理器，在Path里面添加hugo.exe的路径

  1. 接下来就是win+R打开cmd，输入hugo version，如果显示版本号就说明hugo安装成功了

二、搭建博客

2.1 创建博客项目

步骤 1：打开命令行

在D:hugo目录的地址栏输入 cmd，然后按 Enter 键打开命令行窗口。

步骤 2：创建新站点

在命令行中输入hugo new site Blog（将 Blog 替换为你想要的博客名称）：

可以看到你已经新建了一个Blog的文件夹

步骤 3：启动开发服务器

输入cd Blog进入Blog文件夹，接下来输入hugo server -D启动开发服务器

现在打开http://localhost:1313/就可以看到页面，Ctrl+C停止服务

（hugo默认是没有主题的，需要进行主题配置）

2.2 配置主题

Hugo 本身不包含默认主题，我们需要选择并安装一个主题。

步骤 1：选择主题

访问 Hugo Themes 浏览并选择你喜欢的主题。

本教程以 hugo-theme-reimu 主题为例进行演示。

步骤 2：下载主题

  1. 进入主题的 GitHub 页面

  2. 点击 tag →下载合适的版本[reimu版本下载页面](Tags · D-Sketon/hugo-theme-reimu)，下载最新版的

  3. 这里点进Source code下载压缩包，解压下载的文件至Blog目录下的Themes文件夹内，顺便给解压好的文件改个名，注意这里的目录是themes»hugo-theme-reimu;不是themes»hugo-theme-reimu»hugo-theme-reimu

2.3 初始配置

  1. 在Blog目录下创建config文件夹，在config文件夹内创建_default文件夹

  1. 将 theme/hugo-theme-reimu/config/_default/params.yml复制到/Blog/config/_default内

  1. 将 theme/hugo-theme-reimu/data内所有文件复制到/Blog/data中

  1. 将 theme/hugo-theme-reimu/_example内的所有文件复制到/Blog/content中

  1. 将 theme/hugo-theme-reimu/assets内的所有文件复制到/Blog/assets中

  1. 将 theme/hugo-theme-reimu/layouts内的所有文件复制到/Blog/layouts中

  1. 将 theme/hugo-theme-reimu/static内的所有文件复制到/Blog/static中

  1. 复制以下代码至 /Blog/hugo.toml 内
baseURL = 'https://example.com/xxx'
theme = 'hugo-theme-reimu'
defaultContentLanguage = 'zh-CN'
enableEmoji = 'true'
[languages]
[languages.zh-CN]
languageCode = 'zh-CN'
languageName = '简体中文'
weight = 1
hasCJKLanguage = 'true'
title = 'xxxxxxxx'
author = 'xxxxxxxx'

[languages.en]
languageCode = 'en'
languageName = 'English'
weight = 2
hasCJKLanguage = 'true'
title = 'xxxxxxxx'
author = 'xxxxxxxx'

[languages.ja]
languageCode = 'ja'
languageName = '日本語'
weight = 3
hasCJKLanguage = 'true'
title = 'xxx'                                      # 自己编辑
author = 'xxx'                                     # 自己编辑

[markup.highlight]
guessSyntax = 'true'
noClasses = 'false'

[markup.goldmark.extensions.passthrough]
enable = 'true'
delimiters.block = [["\\[", "\\]"], ["$$", "$$"]]
delimiters.inline = [["\\(", "\\)"], ["$", "$"]]

[outputs]
home = ["Algolia", "HTML", "RSS"]

[outputFormats.Algolia]
baseName = "algolia"
isPlainText = 'true'
mediaType = "application/json"
notAlternative = 'true'

[markup]
[markup.tableOfContents]
  endLevel = 4
  ordered = 'true'
  startLevel = 1
  1. 在/Blog的地址栏输入cmd，呼出命令行，输入 hugo server -D 启动博客，出现地址后，ctrl + 鼠标单击进入，此时，浏览器应该就能顺利显示主题示例界面，此时博客搭建完成

2.4 文章页写法

  1. 在 /Blog 的地址栏呼出命令行，输入 hugo new content post/blog3/index.md

  1. 打开这个index.md，其中的draft可以改成false，这样就不是草稿了

2.5 cover图片改法

如果你想改这张图片，你就在/Blog/data里面的covers.yml里面更改成你本地的图片地址
