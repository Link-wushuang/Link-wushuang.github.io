海外节点搭建教程——VPS+住宅ip（一）（使用claude更加安心）

这个是我自己的一个搭建流程，大家可以参考一下，我也是在youtube或者其他渠道里学习了一下，但是不一定保证是最好的（应该说是新手友好吧），不过对于新手来说我觉得是一次不错的入门体验。

这边本来想全部写完的，发现有点多，还是分篇写吧，这里主要讲VPS的下载和配置以及导入v2rayN让你有海外的ip节点，这样你平时上那种风控不太高的外网是完全够用了，而且流量有3T，价格其实跟买一年的机场节点差不多甚至更便宜，流量还这么多。当然就是网速会差那么一点点比起香港这些，所以下载什么东西最好还是用机场节点比较快。

VPS购买——ReckNerd

这边我是在ReckNerd上买的VPS，然后是买的美区的洛杉矶（离中国近一点就网快一点）；这个买什么地区的可以看看自己的需求，像如果是想要订阅Codex这样的，想买日区比较便宜一点，那就是最好ip跟使用订阅的地址是一致的。

然后这个VPS购买是可以用支付宝的，不需要有海外支付方式也可以，很友好。

购买流程

1.进入ReckNerd官网

网站是这个啊ReckNerd官网，这个是有我的推广注册的，大家可以小小支持一下，因为我也是第一次这样推广注册，流程也不是很清楚，但是包是不会坑大家的，或者说应该有点小福利吧，不过我也不知道有没有。。。

然后进去之后点上面的那个紫紫的，这个是优惠折扣的，对于新手来说也不需要很好的VPS，就当练手了。

2.点击购买

进去之后，下拉有这样的选购，是按年付的，我们买最便宜的20刀那个就够了，新手嘛正常。

然后这个的话VPS其实挺有用的，后续也可以在上面部署一些东西啊什么的。当然如果你已经有机子了那很好了。

3.选项的修改

这个我好像刚刚才看到，之前我是怎么买的忘记了，但是我觉得这个机子可以买洛杉矶的，因为到时候我们买住宅ip也是用洛杉矶的，这样就更快一点。

然后改好之后点这个continue，下一步点这个checkout。

4.填写信息并付款

然后这些东西就自己填一下，填一下自己的信息就行，中国的手机号也可以，这个反正没什么关系。

然后下面的支付方式我们选择支付宝，点击付款。

然后后面出来的东西点个pay now后写个邮箱就可以扫描付款了。

VPS配置教程

之后你就有一个VPS了，接下来的图我是截取了youtube网上的教程。

支付完成之后你就跳转到这个页面，每个月是3个T的流量，根本用不完。

并且你也收到一封邮件，这个需要保存好

配置VPS——finalshell

接下来就需要你本地连接来进行配置，这边我选用的是finalshell来进行ssh连接，比较好操作一点吧，因为其他的我也没用过。。。都是跟着教程来的。

下载finalshell

这边直接进入finalshell官网下载，根据自己的电脑选择，这边我是用window的x64。

下载之后安装好就行。

ssh连接

这里我们打开finalshell，点击那个文件夹图标，然后点击左边第一个，选择ssh连接。

然后进行登录信息填写，这边就是邮件里发来的账号密码。

确定后点击连接。

弹出来的窗口点接受并保存。

然后就显示这样就是成功了。

如果显示无法登录的，这边也有教程，是youtube上的博主的，我大部分的也是看它的来进行搭建的。ssh无法正常登录解决方法

安装面板

接下来我们来安装一下面板，可以进行代理。

1.更新软件(Ubuntu)

apt update -y && apt upgrade -y && apt install sudo curl -y

2.安装3XUI面板

bash <(curl -Ls <https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh>)

这边就输入到下面这个输入框然后回车就行。

这个下载安装面板的时候，这个是输入y。

显示这样就是搭建成功了，然后上面那个标绿的需要好好好好的保存好，最好写在一个记事本或者记录的地方。

然后我们输入x-ui。

然后这边，视频上是23，但是我实际去安装的时候我应该写的是24吧，你们具体看看这个BBR是哪个号码然后输进去。

然后这里输入1。

这边显示successfully就说明可以了，接下来不用管它。

打开面板并配置

接下来我们来打开面板，根据之前记录的内容进行登录。

然后在左边点击入站列表，选择添加入站。

这边的话我们用Vless+TCP+Readlity就好了，虽然我的好像不是这个，但是我忘记我之前是怎么搞的了，这个应该是没用问题的。

这个备注随便填就行，然后协议选择vless，这个端口是随机分配的不用管。

传输选择TCP，其他不用管。

然后这个安全选择Reality，下面的Target和SNI都改成microsoft.com。

然后点击这个Get New Cert来创建公钥私钥，之后点击创建。

然后点击左侧的加号，点击下面的二维码，扫一下或者点击复制一下，导入V2rayN（这个windows用户可以用）或者shadowrocket（这个是苹果用户的）就可以了。

这个左侧的是https的，貌似应该是可以导入clash的，不过由于后面买的住宅ip节点还是需要v2rayN的（我是用v2rayN的，至少不会走错吧），所以我们还是用v2rayN。

然后这样就是VPS的节点就告一段落，也有用cloudflare进行DNS一下的加速和高级操作，这个需要先去买一个域名，然后进行操作，这个我放在后面再讲，现在主要是把完整的流程跑一下。

V2rayN

下载v2rayN

接下来我们要导入这个VPS的节点到v2rayN，就要先下载，是这个网址，你们也可以下载其他版本的。v2rayN下载

这个下拉选择v2rayN-windows-64.zip就行。

剩下的下载就很简单，这边就不演示了。

导入VPS节点

这边打开配置项，用我们面板复制的二维码（这个复制右边那个二维码导进去），点击从剪贴板导入，就拥有一个ip节点了

这个就是上面那个闪电的标志是可以测速的，如果测出来是-1就是有问题的没连上。

然后每一个的话你右键设为活动，那就是把它作为你的代理ip了。

OK，这节就是这么多了，写这个还是有点耗时间的。。。
